Bot Management
s.reload - Reload a specific cog
s.reloadall - Reload all cogs
s.eval - Evaluate Python code
s.guilds - List all guilds the bot is in
s.leaveguild - Make bot leave a specific server
Bot Status
s.setstatus - Set bot status (online/idle/dnd/invisible)
s.setactivity - Set bot activity (playing/watching/listening/competing/streaming)
Global Logging System
s.cmdlogs set - Set global command logging channel
s.dmlogs set - Set global DM logging channel
s.guildlogs set - Set global guild join/leave logging channel
s.cmdlogs status - Check command logging status
s.dmlogs status - Check DM logging status
s.guildlogs status - Check guild logging status
Note: All logs go to your bot server globally
System Management
s.cooldown - Set global command cooldown (supports 1s, 5m, 2h, 1d)
s.maintenance - Toggle maintenance mode (complete silence for others)
s.shutdown - Safely shut down the bot
s.clearslash - Remove all slash commands from Discord
Server Stats & Monitoring
s.serverstats start - Start server stats monitoring
s.serverstats stop - Stop server stats monitoring
s.serverstats show - Show current server stats
s.serverstats list - List all monitored servers
s.ratelimit status - Check rate limit status
s.ratelimit setchannel - Set rate limit alert channel
s.ratelimit threshold - Set rate limit threshold
s.ratelimit reset - Reset rate limit warnings
Data Management
s.listdata user - List saved embeds, tickets, reminders for user
s.listdata guild - List auto responders, sticky messages, settings for guild
s.listdata all - Overview of all data: embeds, tickets, transcripts, user data
s.deletedata user - Delete user's saved embeds, tickets, reminders
s.deletedata guild - Delete guild's auto responders, sticky messages, settings
s.resetdata - DANGER Reset all: embeds, tickets, auto responders, user data
Owner Only - Manages: saved embeds, sticky messages, ticket settings, transcripts, user data, auto responders
Communications
s.changelog - Send changelog updates to all server owners
Owner Only - Use for bot changelog updates and version releases
